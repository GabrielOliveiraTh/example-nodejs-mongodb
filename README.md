# Example Node-js MongoDB

This node.js project is an simple example of structure and communication with mongodb.

## Installation

Docker Compose:
```sh
docker-compose up
```

## Endpoints

### [GET] Welcome Route
* http://localhost:9000/

### [GET] List User Route
* http://localhost:9000/users

### [POST] New User Route
* http://localhost:9000/users

```
{
    "name": "Gabriel"
}
```
